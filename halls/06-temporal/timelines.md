# Timelines

## Simple Timeline

```
  ●─────────●─────────●─────────●─────────●
  |         |         |         |         |
 Start     T1        T2        T3       End
```

## Historical Timeline

```
  Past          Present         Future
  ════════════════●═══════════════════→
                  │
                 Now
```
