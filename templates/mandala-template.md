# Mandala Template

Fill in the circles to create your own mandala pattern.

```
      ○   ○   ○
    ○   ○   ○   ○
  ○   ○       ○   ○
    ○   ○   ○   ○
○   ○   ○   ○   ○   ○
  ○   ○   ●   ○   ○
○   ○   ○   ○   ○   ○
    ○   ○   ○   ○
  ○   ○       ○   ○
    ○   ○   ○   ○
      ○   ○   ○
```

## Tips
- Use different characters: ●, ○, ◉, ◎, ⊙, ☉
- Add symmetry by repeating patterns
- Layer circles to create depth
- Consider sacred geometry ratios
