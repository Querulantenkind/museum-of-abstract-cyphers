# Grid/Tessellation Template

Create repeating patterns using grid structures.

## Basic Grid
```
┌───┬───┬───┬───┐
│   │   │   │   │
├───┼───┼───┼───┤
│   │   │   │   │
├───┼───┼───┼───┤
│   │   │   │   │
├───┼───┼───┼───┤
│   │   │   │   │
└───┴───┴───┴───┘
```

## Fill Ideas
1. **Checkerboard**: Alternate filled/empty
2. **Diagonal**: Create diagonal lines
3. **Symbol Pattern**: Place symbols in each cell
4. **Progressive**: Increase complexity left to right

## Example Patterns

┌───┬───┬───┬───┐
