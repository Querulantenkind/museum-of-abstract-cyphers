# Fractal Tree Template

Complete the branching structure to create your fractal tree.

```
        │
        │
        │
    ┌───┴───┐
    │       │
    │       │
  ┌─┴─┐   ┌─┴─┐
  │   │   │   │
 ─┴─ ─┴─ ─┴─ ─┴─
```

## Instructions
1. Start with the trunk (center vertical line)
2. Add main branches at consistent angles
3. Each branch splits into smaller branches
4. Continue subdividing until desired depth
5. Use characters like: │ ─ ┌ ┐ └ ┘ ├ ┤ ┬ ┴ ┼

## Variations
- Symmetric tree: equal angles on both sides
- Asymmetric: vary branch angles and lengths
- Add leaves: ✿ ❀ ✾ ❁ at branch tips
