# Spiral Template

Create your own spiral pattern following these guides.

```
    
    
    
       ●
    
    
    
```

## Spiral Variations

### Archimedean Spiral (equal spacing)
```
        ○
      ○   ○
    ○       ○
  ○           ○
    ○       ○
      ○   ○
        ○
```

### Logarithmic Spiral (expanding)
```
        ○
      ○   ○○
    ○        ○○
  ○             ○○
    ○        ○○
      ○   ○○
        ○
```

## Tips
- Start from center and work outward
- Keep consistent spacing (Archimedean) or expand spacing (Golden Spiral)
- Use rotation: every point should be roughly the same angle from previous
- Characters: ● ○ ◉ ⊙ ☉ ◎ • ∘

## Direction
- Clockwise: natural spiral
- Counter-clockwise: reverse spiral
- Double spiral: two intertwined spirals
