```
    ╔═══════════════════════════════════════════════════════════════╗

    ║  Welcome, Artist.                                             ║
    ║                                                               ║

    ║  where meaning emerges from arrangement, where constraint     ║

### What We're NOT Looking For

```
    ┌──────────────────────────────────────────────────────────────┐
    │                                                              │
    │  ✗ LITERAL REPRESENTATIONS                                   │
    │    Pictures of cats, dogs, faces, objects                   │
    │                                                              │

## [04] TECHNICAL REQUIREMENTS

```
    ┌──────────────────────────────────────────────────────────────┐
    │  CHARACTER SET                                               │
    │  ═════════════                                               │
    │                                                              │
    │  Use primarily ASCII and Box Drawing characters:            │
    │                                                              │
    │  • Basic ASCII: ○ ● - = / \ | _ ~ + * . : ; ' "            │
    │  • Box Drawing: ─ │ ┌ ┐ └ ┘ ├ ┤ ┬ ┴ ┼ ═ ║ ╔ ╗ ╚ ╝          │
    │  • Extended: ╱ ╲ ╳ ◆ ◇ ◈ ○ ● ◐ ◑ ▪ ▫ ▸ ▹ △ ▽ ◁ ▷ ⟡ ⟁        │
    │  • Shading: ░ ▒ ▓ █ ▀ ▄                                     │
    │  • Math: ∞ ∿ ≋ ≈ ≠ ± ∓ × ÷ √ ∑ ∏ ∫                          │
    │  • Arrows: ← → ↑ ↓ ↔ ↕ ↖ ↗ ↘ ↙ ⇄ ⇅ ⇆ ⇋ ⇌                    │
    │                                                              │
    ├──────────────────────────────────────────────────────────────┤
    │  DIMENSIONS                                                  │
    │  ══════════                                                  │
    │                                                              │
    │  Recommended widths:                                         │
    │  • Small pieces: 40-60 characters                            │
    │  • Medium pieces: 60-80 characters                           │
    │  • Large pieces: 80-100 characters                           │
    │  • Epic pieces: 100+ characters (use sparingly)              │
    │                                                              │
    │  Height: No hard limit, but consider scrolling              │
    │                                                              │
    ├──────────────────────────────────────────────────────────────┤
    │  FILE FORMAT                                                 │
    │  ═══════════                                                 │
    │                                                              │
    │  • Plain text (.txt) or Markdown (.md)                       │
    │  • UTF-8 encoding                                            │
    │  • Unix line endings (LF) preferred                          │
    │  • No trailing whitespace                                    │
    │  • Consistent indentation (spaces, not tabs)                 │
    │                                                              │
    └──────────────────────────────────────────────────────────────┘
```

## [05] HOW TO SUBMIT

```
    ╔═══════════════════════════════════════════════════════════════╗
    ║                                                               ║
    ║  SUBMISSION PROCESS                                           ║
    ║  ══════════════════                                           ║
    ║                                                               ║
    ║  STEP 1: FORK                                                 ║
    ║  ───────────────                                              ║
    ║                                                               ║
    ║    Fork the repository to your GitHub account                ║
    ║                                                               ║
    ║         github.com/Querulantenkind/museum-of-abstract-cyphers║
    ║                                                               ║
    ║         ┌──────────┐                                          ║
    ║         │ [Fork] ▸ │                                          ║
    ║         └──────────┘                                          ║
    ║                                                               ║
    ║  ─────────────────────────────────────────────────────────    ║
    ║                                                               ║
    ║  STEP 2: CLONE                                                ║
    ║  ───────────────                                              ║
    ║                                                               ║
    ║    Clone your forked repository locally                      ║
    ║                                                               ║
    ║    $ git clone https://github.com/YOUR-USERNAME/...          ║
    ║    $ cd museum-of-abstract-cyphers                           ║
    ║                                                               ║
    ║  ─────────────────────────────────────────────────────────    ║
    ║                                                               ║
    ║  STEP 3: CREATE BRANCH                                        ║
    ║  ───────────────────────                                      ║

    ║                                                               ║
    ║    $ git checkout -b add-fractal-pattern                     ║
    ║                                                               ║
    ║  ─────────────────────────────────────────────────────────    ║
    ║                                                               ║
    ║  STEP 4: ADD YOUR ART                                         ║
    ║  ──────────────────                                           ║
    ║                                                               ║

    ║                                                               ║

## [06] REVIEW PROCESS

```
    ┌──────────────────────────────────────────────────────────────┐
    │                                                              │
    │  WHAT HAPPENS AFTER SUBMISSION                               │
    │  ═══════════════════════════                                 │
    │                                                              │
    │  1. CURATOR REVIEW                                           │
    │     ────────────────                                         │
    │     Museum curators will examine your submission             │
    │     Timeline: Usually within 48-72 hours                     │
    │                                                              │
    │                 ┌─────────────┐                              │
    │                 │  YOUR ART   │                              │
    │                 └──────┬──────┘                              │
    │                        ↓                                     │
    │                 ┌─────────────┐                              │
    │                 │  REVIEW     │                              │
    │                 └──────┬──────┘                              │
    │                        │                                     │
    │            ┌───────────┼───────────┐                         │
    │            ↓                       ↓                         │
    │     ┌──────────────┐       ┌──────────────┐                 │
    │     │  ACCEPTED    │       │  FEEDBACK    │                 │
    │     └──────┬───────┘       └──────┬───────┘                 │
    │            ↓                       ↓                         │
    │     ┌──────────────┐       ┌──────────────┐                 │
    │     │  MERGED      │       │  REVISIONS   │                 │
    │     └──────────────┘       └──────────────┘                 │
    │                                                              │
    │  ─────────────────────────────────────────────────────────   │
    │                                                              │
    │  2. EVALUATION CRITERIA                                      │
    │     ────────────────────                                     │
    │                                                              │
    │     ☑ Aesthetic Quality                                      │
    │       Does it look good in monospace?                        │
    │                                                              │
    │     ☑ Conceptual Clarity                                     │
    │       Is there a clear idea behind it?                       │
    │                                                              │
    │     ☑ Technical Execution                                    │
    │       Proper formatting, alignment, characters?              │
    │                                                              │
    │     ☑ Originality                                            │
    │       Fresh perspective or approach?                         │
    │                                                              │
    │     ☑ Abstractness                                           │
    │       Avoids literal representation?                         │
    │                                                              │
    │     ☑ Universality                                           │
    │       Understandable across contexts?                        │
    │                                                              │
    │     ☑ Theme Fit                                              │
    │       Belongs in chosen hall?                                │
    │                                                              │
    │  ─────────────────────────────────────────────────────────   │
    │                                                              │
    │  3. POSSIBLE OUTCOMES                                        │
    │     ──────────────────                                       │
    │                                                              │
    │     ✓ ACCEPTED & MERGED                                      │
    │       Your art joins the permanent collection!               │
    │       You'll be credited as contributor                      │
    │                                                              │
    │     ◐ ACCEPTED WITH MINOR CHANGES                            │
    │       Small tweaks needed (alignment, formatting)            │
    │       Curator may make changes directly                      │
    │                                                              │
    │     ◑ NEEDS REVISION                                         │
    │       Good concept, needs technical improvements             │
    │       Feedback provided for resubmission                     │
    │                                                              │
    │     ○ MOVED TO DIFFERENT HALL                                │
    │       Better fit in another category                         │
    │       Will be relocated with your approval                   │
    │                                                              │
    │     ✗ NOT ACCEPTED                                           │
    │       Doesn't fit museum's aesthetic/criteria                │
    │       Detailed explanation provided                          │
    │       Encouraged to submit different work                    │
    │                                                              │
    └──────────────────────────────────────────────────────────────┘

## [08] COMMON PITFALLS

```
    ╔═══════════════════════════════════════════════════════════════╗
    ║                                                               ║
    ║  AVOID THESE MISTAKES                                         ║
    ║  ═══════════════════                                          ║
    ║                                                               ║
    ║  ✗ PROPORTIONAL FONT TESTING                                  ║
    ║    Always test in monospace! Proportional fonts break ASCII  ║
    ║                                                               ║
    ║  ✗ TAB CHARACTERS                                             ║
    ║    Use spaces only. Tabs render inconsistently               ║
    ║                                                               ║
    ║  ✗ TRAILING WHITESPACE                                        ║
    ║    Clean up extra spaces at line ends                        ║
